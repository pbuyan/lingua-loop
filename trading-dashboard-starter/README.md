# Trading Dashboard Starter

Starter front-end architecture for a financial/trading dashboard built with React, TypeScript, TanStack Query, Zustand, AG Grid, and a feature-driven folder structure.

## Getting started

1. Copy `.env.example` to `.env`
2. Install dependencies
3. Run the dev server

```bash
npm install
npm run dev
```

## Notes
